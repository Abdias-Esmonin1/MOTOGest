# Guide utilisateur — MotoGest v1.0

## Utilisation quotidienne

### 1. Saisie des paiements (chaque jour)
1. Cliquer sur **Saisie du jour** dans la barre latérale
2. Vérifier/changer la date si nécessaire
3. Pour chaque moto, saisir le montant versé
4. Sélectionner le statut : Payé / Partiel / Absent / Panne
5. Cliquer **Enregistrer les paiements**

### 2. Déclarer une panne
1. Aller dans **Pannes** → Déclarer une panne
2. Sélectionner la moto concernée
3. Remplir le type et la description
4. La moto passe automatiquement en état "Panne"

### 3. Marquer une panne comme réparée
1. Aller dans **Pannes** → Modifier
2. Changer le statut à **Réparée**
3. Saisir la date de réparation et le coût
4. La moto repasse automatiquement en état "Active"

### 4. Suivre les finances
- **Finances** : voir recettes, dépenses, bénéfice par période
- **Répartition** : voir la recommandation 35/25/20/20
- Enregistrer chaque dépense dans **Enregistrer dépense**

### 5. Exporter un rapport
1. Aller dans **Rapports & Exports**
2. Choisir la période (optionnel)
3. Cliquer **Télécharger Excel** ou **Télécharger PDF**
